## Usage

### Sorting

An array of objects, where each object defines the column key and the sorting order.

```js
[{ key: "calories", order: "asc" }]
```

### Column Search

Enables column specific searching, where the header's **searchable property** is true.

```js
{ key: "calories", title: "Calories", searchable: true }
```

### Column Alignment

Aligns table column based on each header's **align property**, defaulting to left alignment if unspecified.

```js
{ key: "calories", title: "Calories", align: "right" }
```

### Column Width

Sets the width of the table column based on each header's **width property**, defaulting to auto if unspecified.

```js
{ key: "calories", title: "Calories", width: 200 }
```

### Row Styling

Applies bold styling to rows where the item's **bold property** is true.

```js
{ name: "Artichoke", calories: 47, fat: 0.2, carbs: 10.5, protein: 3.3, iron: 1.3, bold: true }
```

### Clickable Rows

Enables row click functionality by listening to the **click:row** event.

```html
<wra-data-table :headers="headers" :items="items" @click:row="handleRowClick" />
```

```js
methods: {
  handleRowClick(row) {
    console.log("Row clicked:", row);
  }
}
```

<Controls />

`caption` the caption text displayed above the table.

`headers` the array of header definitions for the table columns - each header should be an object with at least a `key` and `title` property.
