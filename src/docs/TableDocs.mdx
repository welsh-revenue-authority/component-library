## Usage

### Column Alignment

Aligns table column based on each header's **align property**, defaulting to left alignment if unspecified.

```js
{ key: "calories", title: "Calories", align: "right" }
```

### Column Width

Sets the width of the table column based on each header's **width property**, defaulting to auto if unspecified.

```js
{ key: "calories", title: "Calories", width: 200 }
```

### Row Styling

Applies bold styling to rows where the item's **bold property** is true.

```js
{ name: "Artichoke", calories: 47, fat: 0.2, carbs: 10.5, protein: 3.3, iron: 1.3, bold: true }
```
