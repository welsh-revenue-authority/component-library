# Colour Library

Library of colours for use with the component library. Refer to [tailwind docs](https://tailwindcss.com/docs/colors) for a full list of utilities that use the colour palette.

## Usage

By default, registering the component library imports the CSS variables.

They can be used like this:

`app.css`

```css
/* style sheet classes ~ background colour */

.wra-red {
  background-color: var(--color-wra-red);
}

.wra-light-blue {
  background-color: var(--color-wra-light-blue);
}
```

---

However, to use the colours in Tailwind helper classes like this ...

```html
<!-- html ~ background colours -->

<div class="bg-wra-red"></div>

<div class="bg-wra-light-blue"></div>
```

... the component library's Tailwind stylesheet must be imported.

The instructions are created for Tailwind 4+. The CSS file must be the file imported by `index.html`

`tailwind.css` / `index.css`/ `style.css`,

```css
@import "tailwindcss";
@import "@wra-gov/vue-components/tailwind.css";
```
